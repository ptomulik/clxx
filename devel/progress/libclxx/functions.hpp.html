Details

The detailed progress table shows in detail the implementation status of the basic libclxx functions (C++ wrappers for OpenCL API functions). The OpenCL functions listed in table cover all the API functions defined by OpenCL standard. The meaning of the consecutive columns is the following

FUNCTION original OpenCL function being considered,
C++ whether the C++ wrapper for FUNCTION is implemented,
PYTHON whether the Python wrapper for the C++ function is implemented,
TEST whether a unit test is implemented for the C++ wrapper,
DOC whether API doc (Doxygen) is written for the C++ wrapper.
